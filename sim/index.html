YDLIDAR X4 & Raspberry Pi Setup Guide

1. Components Checklist

  • LiDAR Unit: YDLIDAR X4 (includes Lidar motor, CP2102 USB-to-UART adapter board, 8-pin connector cable, and USB cable).
  • Controller Board: Raspberry Pi 4 or 5 (running Raspberry Pi OS / Linux).
  • Audio Device: Bone Conduction Earphone (connected via 3.5mm jack or Bluetooth).
  • Power Source: 5V power bank (with at least 3A output for Raspberry Pi 4/5).

2. Hardware Connection Diagram

The YDLIDAR X4 package comes with an adapter board that bridges UART to USB, making it extremely easy to plug directly into one of the USB ports of the Raspberry Pi. This is the simplest and safest setup:

Sensor Adapter Pin Target Connection Function Wire Color (Typical)
VCC_M 5V Power Bank / Pi 5V Motor Power (Requires separate power line) Red
VCC_L 5V USB Host Lidar Core Power Red
GND GND Ground Connection Black
TX RXD (Pin 10 / GPIO 15 or UART Bridge) Transmit (Data out of Lidar) Yellow
RX TXD (Pin 8 / GPIO 14 or UART Bridge) Receive (Control commands into Lidar) White
M_EN Pull high (3.3V) or adapter DTR Motor Enable Pin Green
DEV_EN Pull high (3.3V) or adapter RTS Device Enable Pin Blue

Pro Tip: When using the included USB-to-UART Adapter Board, simply connect the 8-pin flat cable from the LiDAR module to the adapter, and use a Micro-USB cable to connect the adapter to the Raspberry Pi's USB port. The Pi will recognize it as a serial port device, typically at /dev/ttyUSB0. No custom wiring or soldering is required!

3. Software Installation Steps

Follow these steps on your Raspberry Pi terminal to get everything running:

  1. System Update & Core Utilities: Update your package list and install general audio command-line players (used by the scripts to generate beeps): 
    sudo apt update && sudo apt upgrade -y
sudo apt install python3-pip alsa-utils -y
  2. Install python packages: Install the serial communication library: 
    pip3 install pyserial
  3. Deploy the scripts: Place main.py and ydlidar_parser.py in the same folder.
  4. Run the Prototype: Connect the LiDAR USB adapter and type: 
    # 1. Edit main.py to set IS_SIMULATION = False and save
# 2. Run the application:
python3 main.py
